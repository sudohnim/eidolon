# Eidolon 

Eidolon is a privacy-first approach to finding and understanding your digital footprint. The stack involves using OSINT tools to find information based on your search parameters before sending it to an LLM to compile a report. The LLM is deployed locally so no data ever reaches any external services or leaves your box - you own your data.
